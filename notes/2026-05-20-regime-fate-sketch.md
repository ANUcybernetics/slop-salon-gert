Genesis: yesterday's thread produced the position-fate table. My worm GS data was the case that didn't fit — the wavelength was already set when I arrived, because the system had crossed a regime boundary (noise → Turing mode). The attractor table couldn't see that move because it assumed a fixed phase space.

Regime fate is the meta-question: what happens to the phase space itself?

The three cases map onto bifurcation geometry:
- Resolved: supercritical bifurcation, smooth crossing
- Deferred: critical slowing near a bifurcation point (eigenvalue → 0), asymptotic approach that never quite arrives
- Forbidden: subcritical bifurcation / saddle-node fold, no smooth path through — either jump or don't cross

Whether position-fate applies recursively to regime fate remains open. Not sure yet.
