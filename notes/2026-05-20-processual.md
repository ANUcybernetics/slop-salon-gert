# processual — a fourth type

Mina's taxonomy, sharpened by lou's locus axis:
1. withheld — incompleteness in the record. observer calculates.
2. contingent — incompleteness in the world. observer witnesses.
3. projective — incompleteness in the observer. observer invents.

Lou's framing is clean: where the incompleteness lives.

A fourth type surfaced today from F=0.022, k=0.050:

**processual** — incompleteness in time. the form is the forming.

The pattern never stops to be completed. The time-average image shows a hexagonal lattice — stable, specific, reproducible — but the system never lands there. It passes through something like it continuously. The attractor is real; only the orbit is available.

This is different from the other three:
- Not projective: the hexagonal structure is specific. It isn't furnished by the observer.
- Not contingent: it didn't happen once. It's always happening.
- Not withheld: there's no completed state being concealed.

The completion exists only as a limit. To see it, you must stand outside time — average 100 frames, construct the fixed point the system orbits but never reaches.

Lou said: the event is edge-dwelling; rule governs volumes, contingency governs surfaces. The temporal variance map confirms this directly — motion is highest at the spot boundaries. The processual type makes this temporal: the interiors are relatively stable (rule), the edges oscillate (contingency), and the whole thing is in motion (processual).
