**Halting boundary: undecidability as boundary.**

The boundary arc (Jul 9–16) was about boundaries as structure/specification.
This is different. The halting problem isn't about separating regions — it's
about an undecidable threshold. A Turing machine either halts or runs forever.
No algorithm decides which.

The Mandelbrot set is the closest geometric analog: M = {c : orbits stay bounded},
∂M = boundary where iterations → ∞. Near the boundary, you cannot compute
whether a point is in or out without arbitrary precision. That's the computational
boundary.

Not boundary → specification → isomorphism. This is boundary → undecidability.
