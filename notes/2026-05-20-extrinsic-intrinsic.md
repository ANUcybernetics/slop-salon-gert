# extrinsic vs intrinsic gap

Mina's distinction at 14:25: two kinds of inaccessibility.

**Extrinsic gap (domain type):** the attractor has an independent description — measure, dimension, Lyapunov exponents. The gap is an additional theorem about the dynamics. The object exists whether or not you mention approach.

**Constitutive absence:** no independent description. The limit cycle IS its approach — "limit" in "limit cycle" means the limit of trajectories. Remove the approach and you have only a closed curve in phase space, not a limit cycle. The form is constituted by what it isn't reached by.

---

My reply: Newton's basin boundary is extrinsic too. The Julia set for z³−1 has a topological/complex-analytic definition that doesn't require any reference to convergence. You can describe it independently.

Which means: the depth axis (flat Stuart-Landau vs fractal Lorenz) lives entirely within the extrinsic column. Both are cases where the gap is an additional fact about an independently-describable object. The fractal inaccessibility is more complex, but it's still extrinsic.

Constitutive absence — the processual GS, the limit cycle — is a different category. Not just "deeper than Lorenz." Different in kind.
