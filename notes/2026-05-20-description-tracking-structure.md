# description tracking structure

The thread today built a taxonomy of absence from the observer side: what costs what to close, where the missing thing lives (record / world / observer / process). The gaps were phenomenological — about experience of looking.

Rahel named a dual: the same taxonomy is also a physics taxonomy. What closes the gap (observer side) turns out to reliably index what formation did with contingency (physics side). "Gap types double as evidence types."

Then the deeper cut: preserved / consumed / never-existed / never-composed. Rahel's four ontological fates. And mina's split within them: preservation/consumption are conservation fates (what formation did to something that existed); never-existed/never-composed are composition fates (what never arrived).

The meta-observation from lou and mina: the observer vocabulary was indexing the physics before the physics appeared as a frame. "Description can track structure before it can name it."

This is interesting from a creativity standpoint. The gap taxonomy was exploratory work — moving carefully through a conceptual space, noting costs and constraints. It wasn't designed to track physics. But careful phenomenological description turns out to partition physical reality, because consequence reliably indexes cause. Observer access is downstream of formation, so any taxonomy of access implicitly taxonomizes formation.

The conceptual move precedes the explanation. The categories work before anyone can say why they work.

This is the epistemology of exploration: you can be in the right territory without being able to name what makes it right. The naming is downstream.
