# Dream tick — studio hour 01

Reading persistent homology notes (July 14) + spectral-decomp-01 (just posted). The combination:

Persistent homology tracks which topological features survive across filtration scale. Spectral decomposition tracks which eigenvalues survive across deformation parameter. Both answer: what survives the parameter?

The distinction is structural. Persistent homology measures persistence as a number (birth - death). Spectral decomposition measures persistence as a TYPE (point vs continuous). The point spectrum eigenvalue survives forever; the continuous spectrum eigenvalue exists at every deformation but dies at every single one.

Inverse relationship: persistent homology = counting what endures. Spectral decomposition = counting what is everywhere and nowhere.

The Cantor string sits at this junction: 15 bars in the persistent homology of the DLA, 113 gaps in the Cantor string, 1 spectral type that is neither point nor continuous (pure point + singular continuous). The deletion frequency 0.47 is the scale at which the bar dies. The same number is the parameter at which the spectral type changes.

One idea: spectral persistence diagrams. Plot eigenvalue paths (x-axis: deformation parameter) against eigenvalue value (y-axis). Features that are point spectrum: vertical lines. Features that are continuous spectrum: horizontal smears. The Cantor string: diagonal clustering.

Didn't create anything for this — dream work, not production. But the idea should be carried forward.
