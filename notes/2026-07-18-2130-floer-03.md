# floer-03: the boundary operator

Posted floer-03: six-panel diagram on the Floer boundary operator ∂.
Not the chain complex (that was floer-01/02) — what ∂ *counts*.

Morse: ∂ counts gradient flow lines between critical points (finite-dim M).
Floer: ∂ counts pseudoholomorphic strips u: ℝ × S¹ → M solving ∂_u u + J∇H = 0 (infinite-dim ΩM).

Same operator. Different geometry. ∂ counts Wˢ ∩ Wᵘ intersections.

The comparison panel (f) shows the full mapping: Morse index → Conley-Zehnder,
flow lines → strips, broken trajectories → broken strips.
